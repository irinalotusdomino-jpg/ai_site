# AI Reels Creator — Hero Block

Односторінковий hero-блок для AI Reels Creator: телефон з реальним reel-відео
в мокапі, приколоті фото навколо, редакційна типографіка (Fraunces + Sora).

## Структура

```
ai-reels-creator/
├── index.html          — весь блок (HTML + CSS + JS в одному файлі)
└── assets/
    ├── logo.webp        — лого
    ├── photo1.webp       — фото 1 (приколоте зліва вгорі)
    ├── photo2.webp       — фото 2 (приколоте зліва внизу)
    ├── photo3.webp       — фото 3 (приколоте справа)
    ├── reel.mp4          — відео в мокапі телефону (без звуку, luped)
    └── poster.webp       — кадр-заглушка відео, поки воно завантажується
```

## Як опублікувати на GitHub Pages

1. Створіть новий репозиторій на GitHub (наприклад `ai-reels-creator`).
2. Завантажте туди всю цю папку (`index.html` + `assets/`), зберігаючи структуру.
3. У репозиторії відкрийте **Settings → Pages**.
4. У полі **Source** оберіть гілку `main` і папку `/ (root)` → **Save**.
5. Через 1–2 хвилини сайт буде доступний за адресою:
   `https://<ваш-логін>.github.io/ai-reels-creator/`

## Як редагувати контент

Усі тексти, кольори, шрифти, посилання на кнопки та шляхи до фото/відео
знаходяться в об'єкті `CONFIG` на початку `<script>` в `index.html`.
Достатньо змінити значення в `CONFIG` — розмітку і CSS чіпати не треба.

Якщо захочете замінити фото або відео — просто покладіть новий файл
в `assets/` з тією самою назвою (або зі своєю, вказавши новий шлях у `CONFIG`).
